# 🔥 Calories Burnt Prediction

A Machine Learning project that predicts the number of calories burnt during exercise using CatBoost Regressor and a Streamlit web application.

## Model Performance

- **Algorithm:** CatBoost Regressor
- **R² Score:** **99.99%**

## Features

- Predict calories burnt based on user inputs
- Fast and accurate predictions
- Simple Streamlit web interface

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- CatBoost
- XGBoost
- Streamlit

## Input Features

- Gender
- Age
- Height
- Weight
- Exercise Duration
- Heart Rate
- Body Temperature

## Output

- Estimated Calories Burnt
